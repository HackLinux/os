[はじめに]
　本ソフトウェアは、独自の組込みOS(カーネル)で開発段階で，1万行程度のライトカーネルです．bootloaderはubootを想定しています．
対象CPUはARM-Cortex-A8で、対象ターゲットは、beagle-borad-xMです。カーネルのAPI仕様は，μITRON4.0，及びT-kerneを参考としています．
ABI仕様は，各種infoを参照しています．
　開発段階という事もあり，デバイスのマクロ定義ファイルは作成していません．


[概要]
  本OSは、OS各機能について，複数のポリシ/メカニズムを搭載しています．
搭載する複数のポリシ/メカニズムは，基本的なOS機能であるタスクスケジューリング、デッドロック問題や優先度逆転問題を回避する戦略を対象としています．さらに，搭載した複数のポリシ/メカニズムを切換えられる機構を設け、動作の検証を可能とするサンプルタスクセットを用意しました．
これにより，本OSを使用し，複数のポリシ/メカニズムの動作を単一環境下から検証する事が可能です．


[野望]
　野望として，カーネルの各機能を極小単位コンポーネントとし，アーキテクチャに留まらず，APIや資源管理機能レベル，カーネルメカニズムやカーネルポリシーレベルでconfig化を可能にする．
現在では、あらゆるデバイスがネットワークに繋がりつつある．使用するハードウェアの制約に合わせて，コンポーネントを動的編成，及び自動化し，このOSをクラウド上で走らせたらおもしろそうだなぁ～＾＾
